Holiday Meal Scheduler Web App
==============================

This is a React-based web app to plan and coordinate meals for a group event.

📦 How to Run Locally:
----------------------
1. Install Node.js (https://nodejs.org)
2. Open a terminal and run:
   npx create-react-app holiday-meal-scheduler
   cd holiday-meal-scheduler
3. Replace the content of src/App.js with the code in this package.
4. Start the app:
   npm start

🌐 How to Publish and Share:
----------------------------
1. Go to https://vercel.com and sign up or log in.
2. Connect your GitHub account and create a new project from this folder.
3. Follow Vercel’s steps to deploy — you'll get a public URL to share.

💡 Features:
- Add/edit guest list
- Assign ingredients by meal and day
- View summary per guest

You can host this on Vercel, Netlify, or any React-friendly hosting service.
